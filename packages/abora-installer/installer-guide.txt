Abora Installer

This launcher opens archinstall with an Abora preloaded configuration.

Preloaded defaults:
- hostname: abora
- audio: PipeWire
- kernel: linux
- locale: en_US.UTF-8 with US keyboard
- packages: plasma-meta, kde-applications-meta, sddm, networkmanager, plasma-nm, konsole, dolphin

You should still review and confirm:
1. Disk layout and partitioning.
2. Bootloader choice.
3. User account and passwords.
4. Timezone.
5. Final package summary before install.

Notes:
- TinyPM is currently guaranteed in the live image. Installed-system packaging can be made fully persistent once the Abora package repo is hosted outside the ISO build.
- If archinstall fails, inspect /var/log/archinstall inside the live environment.
