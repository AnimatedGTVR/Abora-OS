Abora Installer

This launcher opens the Abora installer.

It prefers the Calamares graphical installer when it is present in the image.
If Calamares is unavailable, it can still fall back to the current archinstall backend.

Preloaded defaults:
- hostname: abora
- audio: PipeWire
- kernel: linux
- locale: en_US.UTF-8 with US keyboard
- desktop profile: KDE Plasma with SDDM
- packages: abora-defaults, tinypm-abora, plasma-nm, konsole, dolphin, ark, gwenview, okular

You should still review and confirm:
1. Disk layout and partitioning.
2. Bootloader choice.
3. User account and passwords.
4. Timezone.
5. Final package summary before install.

Notes:
- Abora defaults and TinyPM are preselected for the installed system through the Abora package set.
- If archinstall fails, inspect /var/log/archinstall inside the live environment.
